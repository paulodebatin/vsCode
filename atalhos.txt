CTRL + SHIFT + P = Lista de comandos
CTRL + P = Buscar arquivos
CTRL + B = Contrai/Retrai janela de codigo

CTRL + P em arquivo aberto = Colocar @ no inicio vai navegar pelas tags/simbolos de desenvolvimento
CTRL + SHIFT + L = Multiplos cursores para alteração
CTRL + ALT + Seta = Multiplos cursores para inserção de codigo
CTRL + ' = Abre o terminal
ALT + Seta = Mover codigos para cima ou baixo
ALT + Shift + Seta = Clona o codigo

Ctrl + Shift + K = Excluir linha
Ctrl + Enter = Inserir linha abaixo
Ctrl + Shift + Enter = Inserir linha acima
Ctrl + Shift + I = Reorganizar código no Linux
Shift + Alt + F = Reorganizar código no Windows
